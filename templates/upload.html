CSV File Format

Your CSV file should have the following columns (in any order):

serial_number,location,reading_datetime,vibration,temperature,voltage,current,load_status,fault,year,month,day,hour,minute,second,day_of_week,is_weekend,quarter
TRF-2024-GCM-001,Gicumbi,2024-01-01 00:00:00,2.41,69.84,24956.1,619.4,normal,Normal,2024,1,1,0,0,0,0,False,1
TRF-2024-GCM-001,Gicumbi,2024-01-01 00:00:20,2.46,65.97,25056.1,658.1,normal,Normal,2024,1,1,0,0,20,0,False,1

Required Columns:

  • serial_number - Transformer ID
  • location - Kigali, Bugesera, Rulindo, Gicumbi
  • reading_datetime - YYYY-MM-DD HH:MM:SS
  • vibration - Vibration reading
  • temperature - Temperature in °C
  • voltage - Voltage in V
  • current - Current in A

Optional Columns:

  • load_status - normal, under, over
  • fault - Fault type if any
  • year, month, day, hour, minute, second
  • day_of_week, is_weekend, quarter

Missing optional columns will be auto-calculated

Important Notes:

  • Large files may take several minutes to process
  • Duplicate records (same transformer and timestamp) will be skipped
  • Invalid records will be automatically filtered out
  • Summary tables will be automatically updated after upload
